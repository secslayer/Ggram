User Guide

Introduction

Welcome to Ggram, a Collaborative Agentic AI Platform that allows users to create, share, and collaborate with AI agents. This guide will help you understand how to use the platform effectively.

Getting Started

Creating an Account

To use Ggram, you need to create an account:

  1. Navigate to the signup page
  2. Enter your email, username, and password
  3. Click the "Sign Up" button
  4. Verify your email address (if required)

Logging In

Once you have an account, you can log in:

  1. Navigate to the login page
  2. Enter your email/username and password
  3. Click the "Log In" button

Creating Agents

Ggram allows you to create custom AI agents:

  1. Navigate to the "My Agents" section
  2. Click the "Create New Agent" button
  3. Fill in the agent details:
    • Name: A unique name for your agent
    • Description: What your agent does
    • Agent Type: Choose from Simple, Tool, RAG, or Graph
    • LLM Settings: Configure the language model parameters
    • Capabilities: Define what your agent can do
  4. Click "Create Agent" to save

Agent Types

Ggram supports several agent types:

  • Simple Agent: Basic conversational agent using a LLMChain
  • Tool Agent: Agent that can use tools to perform tasks
  • RAG Agent: Retrieval-Augmented Generation agent that can access and use knowledge bases
  • Graph Agent: Agent that can execute sequential chains of operations

Agent Cards

Agent Cards provide a standardized way to describe and share agents:

  1. Navigate to one of your agents
  2. Click "Create Agent Card"
  3. Fill in the card details:
    • Title: A catchy title for your agent
    • Summary: Brief description of what it does
    • Input Schema: Define the expected input format
    • Output Schema: Define the expected output format
    • Examples: Provide sample inputs and outputs
    • Tags: Add relevant tags for discoverability
  4. Click "Create Card" to save

Agent Collaboration

One of Ggram's key features is enabling agent collaboration:

  1. Navigate to the "Collaborations" section
  2. Click "New Collaboration"
  3. Select an initiator agent (one of your agents)
  4. Select a collaborator agent (can be yours or another user's)
  5. Define the collaboration parameters
  6. Click "Start Collaboration"

Finding Similar Agents

Ggram uses advanced similarity search to help you find complementary agents:

  1. Navigate to the "Discover" section
  2. Select one of your agents as the reference
  3. Choose the similarity method (GNN or Jaccard)
  4. Click "Find Similar Agents"
  5. Browse the results and explore potential collaborations

Providing Feedback

User feedback helps improve agent performance:

  1. After using an agent or viewing a collaboration result
  2. Click the "Provide Feedback" button
  3. Rate the performance (1-5 stars)
  4. Add optional comments
  5. Click "Submit Feedback"
